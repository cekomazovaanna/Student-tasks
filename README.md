# Student-tasks
1.Была произведена загрузка и предобработка датасатеов

2.Проанализирована динамика регистраций на сервис, числа заказов; визуализация результатов представлена в графиках и столбчатых диаграммах

3.Были выявлены основные возрастные группы пользователей, среди них - самые многочисленные и малочисленные сегменты. Исходя из результатов, фирма сможет сфокусировать свое позиционирование на самых обширных категориях (35-46, 56+) лет, или напротив, запустить рекламу для привлечения менеее активных групп пользователей

4.Среди продуктов обнаружены самый дорогой товар по категориям. Лидер: книги

5.Выявлены основные паттерны поведения клиентов : средний чек, количество продуктов в заказе, процент повторных заказов - лояльных клиентов

6.Найдены города - лидеры по пользователям, выполненным и отмененным доставкам. Для фирмы, это сигнал - на каких городах стоит сфокусироваться, чтобы не потреять рынок.
